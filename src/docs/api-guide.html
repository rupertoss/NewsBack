Overview

HTTP verbs

REST notes tries to adhere as closely as possible to standard HTTP and REST conventions in its use of HTTP verbs.

Verb Usage

GET

Used to retrieve a resource

HTTP status codes

RESTful notes tries to adhere as closely as possible to standard HTTP and REST conventions in its use of HTTP status codes.

Status code Usage

200 OK

The request completed successfully

500 Internal Server Error

The request failed, caused by server error or invalid request

Hypermedia

RESTful Notes uses hypermedia and resources include links to other resources in their responses. Responses are in JSON format

Resources

Article

The Article resource is used to show articles returned by user query.

Retrieving articles by country and category

A GET request is used to get articles filtered by country and category

Curl request

$ curl 'http://localhost:8080/news/pl/technology' -i -X GET

Response structure

Path Type Description

country

String

The country of the articles

category

String

The category of the articles

articles

Array

The list of articles

articles[].author

String

The author of the article

articles[].title

String

The title of the article

articles[].description

String

The description of the article

articles[].articleUrl

String

The hyperlink to the article

articles[].imageUrl

String

The hyperlink to the image of the article

articles[].date

String

The published date of the article

articles[].sourceName

String

The source of the article

Example response

HTTP/1.1 200 OK
Content-Type: application/json;charset=UTF-8
Content-Length: 10201
